# CIVITAS Emergency Traffic Coordinator

An emergency traffic coordinator system designed for urban emergency routing, leveraging agents, GCP, and Firebase.

## Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Prajwal20p6/civitas.git
   cd civitas
   ```

2. **Configure Environment:**
   ```bash
   cp .env.example .env
   ```

3. **Explore Modules:**
   - [agents/](file:///e:/civitas/agents): Smart agents for emergency coordination.
   - [backend/](file:///e:/civitas/backend): API and orchestration backend.
   - [frontend/](file:///e:/civitas/frontend): Dashboard and traffic coordinator interface.
   - [deployment/](file:///e:/civitas/deployment): GCP and Firebase deployment configurations.
